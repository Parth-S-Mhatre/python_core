# python_core

A small repository for learning Python — examples, exercises, and notes to help beginners and intermediate learners understand Python fundamentals and practical usage.

## Description

This repository contains curated Python learning materials focused on teaching core concepts and practical skills. It includes:

- Python basics: variables, data types, control flow
- Functions and modules
- Data structures: lists, tuples, sets, dictionaries
- File I/O and exception handling
- Object-oriented programming (classes and inheritance)
- Working with common standard library modules
- Small projects and exercises for hands-on practice

## Contents

- examples/ — short example scripts demonstrating concepts
- exercises/ — practice problems with instructions and tests
- projects/ — small end-to-end projects
- notes/ — concise topic summaries and tips

(If any of these folders are missing, feel free to create them or I can add starter files.)

## Getting started

1. Install Python 3.8+ from https://www.python.org/downloads/.
2. Clone the repository:

   git clone https://github.com/Parth-S-Mhatre/python_core.git
3. Navigate into the project folder and run scripts with:

   python path/to/script.py

## How to use this repo

- Work through examples to understand concepts.
- Complete exercises in the `exercises/` directory and run tests where available.
- Use the `projects/` folder for larger practice assignments.
